# Console Password Manager 
Консольный менеджер паролей на c++, который позволяет сохранять и просматривать учетные данные. данные сохраняются в файл с примитивным шифрованием


# Возможности
- Добавление нового пароля
- Просмотр всех сохраненных паролей
- Простое XOR шифрование
- Сохранение в vault.txt

# Как работает программа
1. Пользователь выбирает действие из меню: добавить парорль, показать все пароли, выход
2. Данные шифруются перед сохранением
3. Пароли записываются в файл vault.txt
4. При просмотре данные расшифровываются автоматически

# Метод шифрования 
Используется простое XOR шифрование
`data[i] ^= 2;`

# Пример работы
```
1. Добавить пароль
2. Показать все
3. Выход
Выбор: 1
Сервис: GitHub
Логин: user
Пароль: 123456

[OK] Данные зашифрованы и сохранены!
```
